# Knowledge Catalog documentation

Use the following resources to generate architectural and design recommendations
for Knowledge Catalog:

*   https://docs.cloud.google.com/dataplex/docs/ai-overview.md.txt: Introduction
    to how Knowledge Catalog supports AI agents and grounding.
*   https://docs.cloud.google.com/dataplex/docs/use-discovery-agent.md.txt:
    Setup guidelines for creating Knowledge Catalog discovery agents using
    python clients or ADK.
*   https://docs.cloud.google.com/dataplex/docs/build-agent-to-enrich-metadata.md.txt:
    Detailed step-by-step guide for creating automated context metadata
    enrichment routines.
*   https://docs.cloud.google.com/dataplex/docs/establish-foundational-data-context.md.txt:
    Tutorial for setting up a business glossary, aspect types, and aspects to
    bind data contexts.
*   https://docs.cloud.google.com/dataplex/docs/enable-policy-compliant-lakehouse-access.md.txt:
    Strategy and guides for setting up access controls and security policies in
    a data lakehouse.
*   https://docs.cloud.google.com/dataplex/docs/use-data-insights-structured-data.md.txt:
    Guide to configure Gemini in BigQuery to generate data summaries.
*   https://docs.cloud.google.com/dataplex/docs/ingest-custom-sources.md.txt:
    Guide to managing metadata for remote or non-Google sources.
*   https://docs.cloud.google.com/dataplex/docs/profile-ensure-data-quality.md.txt:
    Guide to run data scans to verify consistency.
*   https://docs.cloud.google.com/dataplex/docs/use-gemini-cli-agent-to-test-data-context.md.txt:
    Tutorial to set up Knowledge Catalog aspects that provide contextual data,
    and use the Gemini CLI to test the data context and verify that an agent can
    accurately ground its answers in trusted, certified data.
*   https://docs.cloud.google.com/dataplex/docs/lineage-use-cases-impact-analysis.md.txt:
    Guide to track data schema and table relationships to verify system impacts.
*   https://docs.cloud.google.com/dataplex/docs/lineage-use-cases-pii-leakage.md.txt:
    Strategies to trace and isolate sensitive columns.
*   https://docs.cloud.google.com/dataplex/docs/retrieve-data-context.md.txt:
    Guide to use external agents to query Knowledge Catalog metadata catalogs.
